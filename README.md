# TraceDB Protocol

Private staging repository for the TraceDB protocol contract.

This repository owns the canonical HTTP reference, OpenAPI artifact, platform
contract, and executable conformance harness used by the TraceDB core and SDK
repositories during the private organization split.

- `docs/api/v1-openapi.json`
- `docs/platform-contract-v0.md`
- `docs/platform-contract-v0.json`
- `scripts/platform_conformance.py`
